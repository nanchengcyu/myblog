项目简介<br>
此博客前端基于semantic UI 设计，并设配了手机端、pad平板端、网页端浏览器等样式布局
后端基于java框架Spring Boot开发，引入了一下模块：web,Thymeleaf,IPA,MySQL，Aspects，DevTools
其余架构有 mybaits连接MySQL数据库 redis做友链，照片墙等
